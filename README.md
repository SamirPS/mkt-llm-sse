# Gatling Talks and Tutorials

This repository contains the source code for various talks and tutorials about Gatling. Each talk or tutorial is in its own folder, with a `README.md` file explaining the content and how to run the code.

## Getting Started

To get started with Gatling, you can check out the following resources:

- [Gatling Documentation](https://docs.gatling.io/)
- [Gatling GitHub Repository](https://github.com/gatling/gatling)
- [Gatling Community](https://gatling.io/community/)

## Contributing

If you want to contribute to this repository, feel free to open an issue or submit a pull request. We welcome contributions from the community!

 ## Repository Structure
 The repository is organized into the following directories:
    - `articles/`: Contains demo projects that have corresponding how-to guides in the Gatling docs.
    - `meetup/`: Contains demo projects from in-person presentations and talks about Gatling.
    - `webinars/`: Contains demo projects from Gatling webinars.
